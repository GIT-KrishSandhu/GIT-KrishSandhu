## About Me

I'm a B.Tech CSE (AI) student passionate about building production-ready AI systems that combine LLMs, deep learning, and modern software engineering. My interests lie in Agentic AI, Retrieval-Augmented Generation (RAG), Computer Vision, and scalable backend development, with a focus on turning research concepts into practical applications.

Recently, I've been developing multi-agent AI platforms like **Stadium Stan**, an AI-powered stadium operations system, **Data Detective**, an agentic dataset auditing platform built for the Microsoft Agents League Hackathon, and **OceanLens**, a RAG system for querying oceanographic data. I'm currently exploring Azure AI Foundry, Microsoft Fabric, LangGraph, Google ADK, and MLOps to build reliable, production-grade AI solutions.
